# bubitr-relay

## Installation

### requirement

| name   | version |
| ------ | ------- |
| nodejs | 20.5.1  |
| pnpm   | 8.9.0   |

### run locally

```bash
  git clone https://github.com/KingYoSun/bubitr-relay.git
  cd bubitr_relay
  mkdir output
  cp .env.default .env
  docker compose up --build -d
```

## Features

- Light/dark mode toggle
- Responsive
